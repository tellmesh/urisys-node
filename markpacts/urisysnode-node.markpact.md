## Unpack → `.markpact/`

```bash
cd urisys-node
export TELLMESH_ROOT=/home/tom/github/tellmesh

urisys markpact run "$PACK" --as pack
urisys markpact run "$PACK" --as service --port 8789
urisys markpact run "$PACK" --as interface
urisys markpact run "$PACK" --as adapter
```
